## Instructions for Build and Use

Steps to build and/or run the software:

1. Make sure that the Java development kit (JDK) is installed on the system.
2.Place all project files (like SnakeGame.java, GamePanel.java, and Movable.java) in the same directory.
3.Compile the program using javac *.java or open the project in an IDE and run the SnakeGame class.
Instructions for using the software:

1. Launch the program to open the game window.
2. Use the arrow keys (Up, Down, Left, Right) to control the movement of the snake.
3. Move the snake over apples to increase its length.
